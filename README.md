# Lemon.Toolkit

Introduction
##### Lemon.Toolkit
A toolkits collection client based on Semi-Avaloniaui and Lemon.Extensions.SlimModular.
Up to now, It contains:
- File Inspector (ongoing)
- File Comparer (ongoing)
- ...
